## micro:bit Online Editors
1. <a href="https://makecode.microbit.org/" target="_blank">MakeCode Editor</a> – a visual editor which uses blocks to build programs. Good for beginners who want to learn programming logic and abstraction.
2. <a href="https://python.microbit.org/v/3" target="_blank">Python Editor</a> – a text editor for writing programs in MicroPython. Good for continuing users who want to get familiar with real-world programming.
